# Web to Executable - Node.js Bridge

This is the local Node.js application required to connect the "Web to Executable" web interface to your local machine. It acts as a bridge, receiving commands from the web UI to perform local build operations.

## Setup Instructions

### Prerequisites
You must have [Node.js](https://nodejs.org/) installed on your computer (which includes npm).

### 1. Install Dependencies
Open your terminal or command prompt, navigate to this directory (the one containing `package.json`), and run the following command:

```bash
npm install
```
